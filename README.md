# Predict Air Quality Index for Delhi

### Parameters Used - 

* Temperature

* Wind speed

* Relative Humidity

* Traffic index

* Air quality of previous day

* Industrial parameters such as power plant emissions


### Algorithm Used - 

* Support Vector Machine (SVM)
* Naive Bayes
* Other NN algo's
